Qnekt

Qnekt

Qnekt is a fork of Metamask

  • Since Qtum is a fork of Bitcoin and uses UTXOs, transaction signing is completely different from Ethereum.
  • Because of that, it is not possible to just add Janus as a remote endpoint in Metamask and have everything just work.
  • For a more technical understanding, see Why the need to fork Metamask?
